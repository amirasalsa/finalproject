Main idea:
- To create a platform for online lessons through means of videos and written posts.
- To create a more structured organization of said videos and written posts.

Features:
- Register and Login
- Logout
- Select Membership: Free, Enterprise, Professional
- Input Credit Card
- View created Posts
- Create a Post 
- Update a Post
- Delete a Posts
- View Course List
- View Course Lessons
- Play Lesson Video
- View Profile Information
- Update Profile
- Add Profile Image
- Search (in progress)

APIs Used:
- Stripe
